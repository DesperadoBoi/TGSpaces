# TGSpaces

TGSpaces — менеджер отдельных Telegram-клонов для Android. Каждый слот — отдельное приложение Telegram с отдельным входом в аккаунт.

## Возможности

- установка отдельных Telegram-клонов;
- отдельные `packageName` для каждого клона;
- запуск клонов из TGSpaces;
- открытие настроек клонов;
- проверка обновлений через `catalog.json`;
- release-сборки.

## Как установить

1. Перейдите в GitHub Releases проекта.
2. Скачайте `TGSpaces-release.apk` из последнего release.
3. Установите APK на Android-устройство.
4. При необходимости разрешите установку неизвестных приложений.
5. Откройте TGSpaces и установите нужные клоны.

## Как пользоваться

1. Нажмите `Добавить слот`.
2. Нажмите `Установить клон`.
3. Дождитесь загрузки APK.
4. Подтвердите установку в системном установщике Android.
5. Откройте установленный клон.

## Важно

- Debug APK и Release APK могут конфликтовать из-за разных подписей.
- Если раньше стояла debug-версия, ее может потребоваться удалить.
- Если иконка в настройках Android отображается неправильно, откройте клон один раз или перезагрузите телефон.

## Дисклеймер

TGSpaces не является официальным приложением Telegram.

Проект не связан с Telegram Messenger Inc.

Клоны основаны на открытом исходном коде Telegram Android, но распространяются как неофициальные сборки.

## Для разработчика

Подготовка release-комплекта:

```powershell
.\scripts\prepare-release.ps1 -ReleaseTag v0.x-release
```

Подготовка release-комплекта без пересборки клонов:

```powershell
.\scripts\prepare-release.ps1 -ReleaseTag v0.x-release -SkipCloneBuild
```

Каталог клонов генерируется через:

```powershell
.\scripts\update-clone-catalog.ps1
```
