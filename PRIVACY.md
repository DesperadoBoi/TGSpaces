# Политика конфиденциальности

TGSpaces не требует регистрации.

TGSpaces не собирает аккаунты Telegram, сообщения, контакты, медиа, пароли или другие данные пользователя из Telegram-клонов.

TGSpaces использует интернет только для загрузки catalog JSON и APK-файлов из GitHub Releases.

TGSpaces может открывать системный установщик Android для установки APK-файлов.

Данные Telegram-клонов хранятся внутри самих приложений-клонов, а не в TGSpaces. TGSpaces не имеет доступа к содержимому аккаунтов, сообщений, контактов или медиа внутри установленных клонов.

TGSpaces не является официальным приложением Telegram.
